#### Clickable Prototype 

React/Vite кликабельный прототип для проверки продуктовой гипотезы и валидации логики правил. Прототип использовался для проверки сценариев гибридного подхода наполнения витрины: протестировано более 200 комбинаций правил, зафиксировано время выполнения ключевых операций, проверены fallback-сценарии, а также входе помог выявить критические логические пробелы и противоречия. 

Прототип не является финальным интерфейсом и не отражает итоговый UX/UI-дизайн продукта. 
Его цель - проверить бизнес-логику, сценарии и поведение системы в пограничных случаях.
