#### Clickable Prototype 

React/Vite кликабельный прототип для проверки продуктовой гипотезы и валидации логики. Прототип использовался для проверки сценариев гибридного подхода наполнения витрины: помог выявить критические логические пробелы и противоречия, протестировать около 200+ комбинаций правил, зафиксировать время на выполнение ключевых операций, проверить fallback-сценарии. 

Прототип не является финальным интерфейсом и не отражает итоговый UX/UI-дизайн продукта. 
Его цель - проверить бизнес-логику, сценарии и поведение системы в пограничных случаях.
